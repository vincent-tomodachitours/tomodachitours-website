# SEO Optimization Plan for Tomodachi Tours

## Phase 1: Foundation & Technical SEO (Priority: Critical)

### 1.1 Basic Meta Tags & Page Titles
- **Dynamic page titles** for each route
- **Unique meta descriptions** for all pages (150-160 characters)
- **Meta keywords** for each page
- **Canonical URLs** to prevent duplicate content

### 1.2 Essential Files
- **robots.txt** - Guide search engine crawling
- **sitemap.xml** - Help search engines discover pages
- **Updated manifest.json** - Fix PWA metadata

### 1.3 Open Graph & Social Media
- **Open Graph tags** (og:title, og:description, og:image, og:url)
- **Twitter Card tags** for better social sharing
- **Tour-specific images** for social previews

## Phase 2: Content & Structured Data (Priority: High)

### 2.1 Structured Data Implementation
- **LocalBusiness schema** - Business info, location, contact
- **TouristAttraction schema** - For each tour page
- **Review/Rating schema** - Customer testimonials
- **BreadcrumbList schema** - Navigation structure
- **Organization schema** - Company information

### 2.2 Content Optimization
- **H1 tag optimization** - Unique for each page
- **Alt text for images** - Improve accessibility & SEO
- **Internal linking strategy** - Connect related content
- **FAQ sections** - Target long-tail keywords

## Phase 3: Local & Tourism SEO (Priority: High)

### 3.1 Local Business Optimization
- **Google Business Profile** integration
- **Local keywords** - "Kyoto tours", "English tours Kyoto"
- **Location-based content** - Neighborhood guides
- **Contact information** - Consistent NAP (Name, Address, Phone)

### 3.2 Tourism-Specific SEO
- **Tour booking schema** - Pricing, availability, duration
- **Event schema** - For scheduled tours
- **Travel guide content** - Blog/tips section
- **Seasonal content** - Best times to visit

## Phase 4: Performance & Advanced SEO (Priority: Medium)

### 4.1 Technical Improvements
- **Core Web Vitals** optimization
- **Image optimization** - Lazy loading, proper sizing
- **URL structure** - Clean, descriptive URLs
- **404 error handling** - Custom error pages

### 4.2 International SEO
- **Hreflang tags** - For future multi-language support
- **Currency/pricing** - Local formatting
- **Cultural content** - Japan-specific information

## Current SEO Status Assessment

### Critical Issues Found ❌
- Generic meta description: "Web site created using create-react-app"
- No dynamic page titles: All pages share "Tomodachi Tours" title
- Missing Open Graph tags: No social media sharing optimization
- No structured data: Missing schema markup for tours, business info, reviews
- No robots.txt: Search engines have no crawling guidance
- No sitemap.xml: Search engines can't efficiently discover pages
- Generic manifest.json: Still shows "React App" instead of brand

### What's Working ✅
- Clean URL structure: SEO-friendly paths
- Semantic HTML: Good use of headings (h1, h2, h3)
- Mobile responsive: Proper viewport meta tag
- Fast loading: Using WebP images for performance
- Content quality: Rich, descriptive content about tours
- Internal linking: Good navigation between pages
